1]color

  1. Color Property is used to change color of text.
  2. Example:
    This is red color text

2]font-family

  1. Specifies the typeface (font) of the text.
  2. Example:
    This text has Arial font-family

3]font-size

  1. Specifies font-size of the text.
  2. Example:
    This text has 20px font-size

4]font-weight

  1. The font-weight property sets how thick or thin characters in text should be displayed
  2. Example:
    This text has font-weight bold

5]font-style

  1. The font-style property is mostly used to specify italic text
    • normal - the text is shown normally
    • italic - the text is shown in italics
    • oblique - less supported.
  2. Example:
    This text has font style italic.

6]line-height

  1. The line-height property specifies the height of a line.
  2. Example:
    This is a paragraph with a bigger line-height.The line height is here set to 200%.

7]letter-spacing

  1. The letter-spacing property increases or decreases the space between characters in a text.
  2. Example
    this text has letter-spacing: 3px;

8]text-transform

  1. The text-transform property controls the capitalization of text.
    • uppercase:
    • lowercase:
    • capitalize:
  2. Example:
    -This text is Uppercase
    -This text is lowercase
    -This text is capitalize

9]text-decoration

  1. The text-decoration property specifies the decoration added to text
    • overline
    • line-through
    • underline
    • none
  2. Example:
    This text has underline
    This text has overline
    This text has line-through

10]text-align

  1. The text-align property specifies the horizontal alignment of text in an element.
    • center
    • left
    • right
    • justify
  2. Example:
    3. text-align: center;
    text-align:right;

11]text-shadow

  1. text-shadow property adds shadow to text
  2. text-shadow: h-shadow v-shadow blur-radius color;
  3. Example:
    This text has shadow

12]word-spacing

  1. Specify the space between words.
  2. Example:
    This text has word-spacing of 30px

13]text-indent

  1. The text-indent specifies the indentation of the first line in the text-block
  2. Example:
    Lorem ipsum dolor sit amet consectetur adipisicing elit. Doloremque aliquam est eum eos corporis excepturi, veniam molestiae, incidunt possimus labore nobis blanditiis quo. Quam tempora deleniti repellendus sint obcaecati optio?

14]white-space

  1. The white-space property specifies how white-space inside an element is handled.
    • normal: default
    • nowrap: collapse spaces but text will not wrap
    • pre: Preserve spaces and newline,no wrapping.
    • pre-wrap: preserve spaces and new line but wrap text.
    • pre-line: Collapses spaces, and newlines, wraps text
  2. Example:
    This is and text with property white-space: pre-wrap;

15]word-wrap

  1. The word-wrap property allows long words to be able to be broken and wrap onto the next line
    • normal: default
    • break-word: Allows unbreakable word to be broken.
  2. Example:
    Thisisalongwordwehaveusepropertyword-wrap:break-word

16]text-overflow

  1. The text-overflow property specifies how overflow content that is not displayed should be signaled to the user.It can be clipped,diaplay an ellipsis(...) or display a custom string
    • clip: default
    • ellipsis: Render an ellipsis(...) to represent the clipped text
  2. Example:

    text-overflow: clip;

    This is clip text

    text-overflow: ellipsis;

    This has ellipsis text
